Other things

There is an abandoned re-write of Space Cat Games called Space Cat Games Version II. It was maintained for a short period of 2 months, and was released once. It never went mainstream, and never replaced Space Cat Games Version 11 (latest at the time). There is also a lost version of Space Cat Games Version 11, that was known internally as "SPC Legacy". It was to be released when Space Cat Games II went mainstream. It was never released, however, functions relating to SPC Legacy are still present, for example, "func.checklegacywarn()" are still used for checking what version the server is running, and to warn if running an outdated version. Originally, "func.checklegacywarn()" was going to be used to check if the version was the legacy version. If it was true, it would warn the user that the legacy version is not maintained. Now it has been demoted to a version checker. Poor guy.
